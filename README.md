# amicus-sdk

The open client + docs for **amicus**, the hosted legal-research service over the
open [`legalize-pe`](https://github.com/crafter-research/legalize-pe) corpus.

- **`apps/cli`** — `@crafter/amicus`: CLI + MCP server. Search norms, fetch full
  text, and `ask` grounded/cited legal questions. `npx @crafter/amicus`.
- **`apps/docs`** — Fumadocs site (corpus + API + MCP/CLI reference).

amicus (the hosted product app) is a separate, private repo. legalize-pe (the
corpus) is open. This repo is the open, amicus-branded client surface.
